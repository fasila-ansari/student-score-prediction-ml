# student-score-prediction-ml
Machine learning project predicting student scores based on study hours
## Overview
This project predicts a student's exam score based on the number of hours studied using a simple Linear Regression model implemented in Python.
## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
## How It Works
The model is trained on a dataset containing study hours and corresponding exam scores.
The user can enter the number of hours studied, and the program predicts the expected exam score.
## Example
Input: 6 hours
Predicted score: 73
## Purpose
This project was developed to practice basic machine learning concepts and improve practical programming skills using Python.
